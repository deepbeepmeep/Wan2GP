This directory preserves the audit trail for the Wan2GP multi-sprint migration (Sprints 1–4, April 2026). Each sprint subdirectory contains consumed orchestrator handoff artifacts — staged file manifests, handoff notes, working-tree patches, and any untracked-additions bundles — that were useful during commit orchestration but are not part of day-to-day development. Read these only when reconstructing migration history, verifying how a sprint commit was assembled, or auditing a discrepancy after the fact. The live work products (drift CI workflow, rebase runbook, patch lifecycle tests) live under `.github/workflows/`, `docs/`, `source/`, and `tests/` respectively.
